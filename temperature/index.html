Obiettivo

Si intende studiare la temperatura media stagionale per un appartamento nella città di Brescia.

Procedimento

A grandi passi, si intende seguire questo procedimento:
  1. Raccolta dei dati;
  2. Elaborazione (calcolo della media) per ogni stazione;
  3. Interpolazione spaziale;
  4. Documentazione.

Problema: interpolazione

Si dispone di dati limitati (i dati non sono disponibili appartamento per appartamento): è quindi necessario fare un'interpolazione nello spazio.

Problema: calcolo della media

Si dispone di circa 100k datapoint, e si vuole determinare un unico valore di temperatura, noto come "media stagionale".

Formalizzazione

Date N tuple \[(\text{Tempo}\in\mathbb N, \text{Temperatura}\in\mathbb R, \text{Posizione}\in\mathbb R^2)\] si vuole ricavare un campo scalare \[T_{media} = \mathbb R^2\rightarrow\mathbb R\]

Raccolta dati

I dati sono stati richiesti tramite la piattaforma Web dell'ARPA Lombardia.

Per ottenere la massima precisione si è deciso di raccogliere i dati per le due centraline di Brescia (Pastori e Ziziola) su base suboraria, per minimizzare gli errori di discretizzazione.

Errori di discretizzazione

Si definisce errore di discretizzazione quello dovuto alla conversione di una funzione continua a valori continui \[f:\mathbb R\rightarrow S\] in una discreta a valori discreti \[g:\mathbb N\rightarrow S'\] (con \(S'\) discreto).

Calcolo della media stagionale

Cerchiamo una funzione che dati N rilevamenti \(x\in\mathbb R\) produca un valore medio \(x'\in\mathbb R\)\[media:\mathbb R^N\rightarrow\mathbb R\]

Tradizionalmente, si usano queste medie:

  • Aritmetica, \(\frac 1n\sum{i=0}^nx_i\);
  • Geometrica, \(\sqrt[n]{\prod_{i=0}^nx_i}\)
  • Armonica, l'inverso della media dei reciproci.

Scelta della media

Si sa che la media geometrica attribuisce molto più peso ai valori più piccoli: questa è una caratteristica non desiderabile. Lo stesso vale per la media armonica.

Scegliamo quindi la media classica (aritmetica):\[media({x_0,x_1,\dots x_n})=\frac 1n\sum_{i=0}^nx_i\]

Interpolazione spaziale

Scegliamo di dare a ciascuna data source un peso inversamente proporzionale al quadrato della distanza (come l'intensità di una radiazione): il peso è un campo scalare \[P(x,y,z)=\frac{1}{\Delta s^2}=\frac{1}{(\Delta x)^2+(\Delta y)^2+(\Delta z)^2}\]

Condizionamento

Il valore \(\lim_{\Delta s\rightarrow 0} P(x,y,z)=+\infty\) è indesiderabile: per ovviare a questo problema aggiungiamo un fattore costante \(c\) al denominatore, in modo che \(\lim_{\Delta s\rightarrow 0} P = c\). \[ P(x,y,z)=\frac{1}{\Delta s^2+c}=\frac{1}{(\Delta x)^2+(\Delta y)^2+(\Delta z)^2+c} \]
