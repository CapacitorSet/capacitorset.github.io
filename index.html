My projects

Box.js

A tool for studying malware (droppers) written in JavaScript and JScript. It makes use of the Node.js sandbox and provides a dummy implementation of some key ActiveX components (XMLHTTP, ADODB.Stream and others). Used successfully by security researchers and malware analysis services (Sekoia), and included in Lenny Zeltser's REMNux and ArchStrike.

malware analysis, node.js

Rebridge

A completely transparent bridge between JavaScript and Redis, as a Node.js package (rebridge on npm).
